# Trellis Syntax

Syntax highlighting for Trellis sidecar specification files (`.trellis`) in
VS Code-compatible editors.

This extension is packaged from the `editors/vscode` directory:

```sh
vsce package
```

Install the generated `.vsix` with:

```sh
code --install-extension trellis-syntax-0.1.1.vsix
cursor --install-extension trellis-syntax-0.1.1.vsix
```
